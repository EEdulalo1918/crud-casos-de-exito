# CRUD - Panel de Administración

Sistema de administración web desarrollado con PHP, JavaScript, HTML y CSS,
conectado a una base de datos MySQL mediante PDO.

## Módulos
- Casos de Éxito
- Noticias
- Eventos
- Beneficios
- Categoría Beneficios
- Bolsa Universitaria
- Recuerdos
- Comentarios

## Requisitos
- XAMPP instalado (PHP 7.4+ y MySQL)
- Navegador moderno

## Instalación
1. Clonar el repositorio dentro de `xampp/htdocs/`
```bash
   git clone https://github.com/TU_USUARIO/TU_REPO.git
```
2. Importar `database.sql` en phpMyAdmin
3. Copiar `conexion.example.php` y renombrarlo a `conexion.php`
4. Editar `conexion.php` con tus credenciales de MySQL
5. Acceder desde el navegador: `http://localhost/CRUD`

## Tecnologías
- PHP (PDO)
- JavaScript (AJAX)
- HTML5 / CSS3
- MySQL
- XAMPP
